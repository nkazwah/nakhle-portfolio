# Nakhle Kazwah - Portfolio Website

A stunning, modern portfolio website built with Next.js 14, React 18, and Tailwind CSS.

## Features

- **Responsive Design** - Beautiful on all devices
- **Dark Mode** - Enterprise-meets-edgy aesthetic with dark background
- **Smooth Animations** - Glassmorphism effects and gradient accents
- **Sections Included**:
  - Hero section with call-to-action
  - About me with skills showcase
  - Career journey with expandable timeline
  - Portfolio section (ready for future projects)
  - Footer with social links

## Tech Stack

- **Next.js 14** - React framework with App Router
- **TypeScript** - Type safety
- **Tailwind CSS** - Utility-first styling
- **Client-side Components** - Interactive features

## Getting Started

### Installation

```bash
npm install
```

### Development

```bash
npm run dev
```

Visit [http://localhost:3000](http://localhost:3000) to view the site.

### Production Build

```bash
npm run build
npm start
```

## Project Structure
